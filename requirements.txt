# ==================== 安装说明 ====================
# 
# 【基础安装】运行核心功能所需：
#   pip install -r requirements.txt
#
# 【最小安装】仅命令行运行（无Web UI）：
#   pip install ultralytics opencv-python numpy pandas matplotlib pyyaml tqdm
#
# 【完整安装】包含所有可选功能：
#   pip install -r requirements.txt --upgrade
#
# 【GPU环境】确保已安装CUDA和cuDNN：
#   - CUDA 11.8 或 12.1
#   - cuDNN 8.x
#   验证：python -c "import torch; print(torch.cuda.is_available())"
#
# 【常见问题】
# 1. OpenCV安装失败：
#    - Windows: pip install opencv-python-headless
#    - Linux: sudo apt-get install libgl1-mesa-glx
#
# 2. PyTorch CUDA版本不匹配：
#    访问 https://pytorch.org/get-started/locally/ 选择对应版本
#    例如：pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
#
# 3. 内存不足：
#    - 降低 batch size
#    - 使用更小模型（YOLOv8n替代YOLOv12）
#    - 启用混合精度训练（AMP）
